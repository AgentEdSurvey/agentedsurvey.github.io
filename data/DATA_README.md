# Reconstructed 101-Study Corpus

This data release was reconstructed from the submitted manuscript and the
candidate exports supplied by the authors.

## Files

- `full_corpus_101.csv`: 101 bibliographic records.
- `full_corpus.bib`: BibTeX for the 101 records.
- `coding_matrix_full.csv`: conservative draft coding based only on explicit
  title/abstract evidence.
- `corpus_statistics.csv`: bibliographic statistics plus conservative coding
  counts.
- `selection_audit.csv`: source and matching audit for each retained record.
- `coding_manual.md`: coding definitions and decision rules.

## Important status note

The first 53 records are studies explicitly cited in the submitted manuscript.
The remaining 48 were selected from the supplied RIS candidate pool because
their titles and indexed abstracts explicitly describe an LLM/generative-agent
design in psychology, behavior, education, or human-agent interaction.

This reconstruction must not be represented as the original final corpus
without author verification. The coding matrix is intentionally conservative:
blank cells mean that a label could not be safely inferred from the title and
indexed abstract. Full-text verification and independent reliability checking
are still required before the coding statistics are used in the manuscript.
